Descrevendo algumas tags HTML* de formatção de texto

Tag <em>

Esta tag é utilizada pra dar ênfase em uma página HMTL. O seu comportamento padrão exibe o texto em ítalico.

Tag <strong>

Essa tag é utilizada para dar importâcia ao texto. Tendo como o padrão o negrito .

Tags <sup> e <sub>

Estas tags são utilizadas para definir texto sobrescrito e sobrescrito. Sendo muito utilizadas em fórmulas e expressões matemáticas. Exemplo: 27

Tags <ins> e <del>

Estas tags são utilizadas para deixar explícitas inserções e remoções de texto. Muito utilizadas utilizada em blogs, quando o autor quer mostrar alguma atualização no texto.

Tag <mark>

Esta tag é utilizada para marcar trechos do texto. Possibilitando marcar palavras específicas ou trechos maiores.
