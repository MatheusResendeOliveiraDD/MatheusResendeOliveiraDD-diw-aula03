Pesquisa sobre o uso de algumas tags

Tag <adress>

É usada para fornecer ao leitor formas de entrar em contato, etc. Lembrando que o contéudo da tag é exibido em ítalico. Exemplo:

Matheus Email
Endereço: Rua das araras, 10000
Telefone: 31 7070-7070

Tag <blockquote>

Esta tag é utilizada para indicar longas citações dentro de um documento HTML.

Com o uso desta tag os navegadores geralmente recuam o seu conteúdo.

Tag <code>

É usada para marcar o texto inserido como sendo um código de computador. Exemplo: variavel= x

Tag <dfn>

A tag Definition Element é utilizada identificar ou dar destaque à um espaço pontual dentro de um texto. O conteúdo é apresentado em ítalico

Tag <pre>

Com essa tag, conhecida como: texto pré-definido, é possível inserir um conteúdo para ser visualizado da maneira como foi editado. Exemplo: 

    background:#00000; 
    color:#FFFF;

Tag <span>

É utilizado para formatar pequenos textos. Porém se não utilizado juntamente com o CSS, o texto não será alterado. Visto que dentro da tag é possível alterar os estilos, através do CSS.

Tag <time>

É usada para representar datas precisas ou não. Valendo lembrar que não é suportado por alguns navegadores. Entrega do exercício
